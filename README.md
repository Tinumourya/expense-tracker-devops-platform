# Expense Tracker DevOps Platform

## Overview

This project demonstrates a complete cloud-native DevOps implementation for an open-source MERN Expense Tracker application.

The original MERN application is based on:
https://github.com/AjaySingh470/Expense-Tracker-Mern

My contribution focused on designing, deploying, monitoring, and documenting the complete DevOps infrastructure around the application.

---

## Technologies

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Docker
- Docker Hub
- Terraform
- AWS
- Amazon EKS
- Kubernetes
- Jenkins
- Prometheus
- Grafana

---

## Features

- Dockerized frontend and backend
- Infrastructure provisioning using Terraform
- Amazon EKS cluster deployment
- Kubernetes manifests
- Jenkins deployment
- Monitoring using Prometheus and Grafana
- Professional documentation
- Architecture diagrams
- Troubleshooting guide
